文本

通过文本属性，您可以改变文本的颜色、字符间距，对齐文本，装饰文本，对文本进行缩进，等等。

（我是一个正常缩进的P标签还居中了）诗歌分为古体诗和近体诗。古体诗最早的形式是四言古诗。醇厚古朴，简质典雅，主要在春秋战国时期。之后发展为五言古诗，主要为汉乐府诗。之后就是七言古诗了，而它的创始人是曹丕。在此之后就进入到近体诗了。最早是五言侓诗，之后出现了七言律诗、五言绝句和七言绝句。总体评价的话，古体诗比较自由随性，而近体诗尤其到唐宋时代，格律平仄则是很重要的一点。唐诗宋词作为近体诗的代表，值得我们细细品味。（详情可见文章《中国诗歌艺术研究》。）
