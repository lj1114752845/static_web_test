    id选择器

    id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。

    我是一个么得感情的p标签

    class选择器

    class 选择器可以为标有特定 class 的 HTML 元素指定特定的样式。

    我是一个么得感情的p标签

    属性选择器

    可以为拥有指定属性的 HTML 元素设置样式，而不仅限于 class 和 id 属性。

    我是一个么得感情的p标签

    我是第二个么得感情的p标签

    继承

    子标签会继承父标签的属性 子标签可以有自己的属性来重写（我认为这属于重写操作）父标签的属性

    我是父类

    我继承父类的属性

    我不继承父类字体颜色的属性
